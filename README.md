# ✝ EmmausAI — AI-Aided Bible Education for All People

> "Did not our hearts burn within us while He talked with us on the road?" — Luke 24:32

**EmmausAI** is a free, open-source, AI-powered Bible education platform for all people under God/Jesus.  
Walk the Emmaus road — learn Scripture through AI tutoring, structured courses, daily devotionals, and community.

## 🌟 Inspired By

- **[SilverConnect](https://silverconnect.vercel.app)** — Free technology architecture
- **[GraceCovenant](https://gracecovenant.vercel.app)** — Christian community platform

## 🛠 Free Tech Stack

| Layer | Technology | Cost |
|-------|-----------|------|
| Framework | Next.js 15 + React 19 | Free |
| Language | TypeScript | Free |
| Styling | Tailwind CSS + Radix UI | Free |
| Database | Supabase (PostgreSQL) | Free tier |
| Auth | Supabase Auth (SSR) | Free tier |
| ORM | Drizzle ORM | Free |
| AI | Vercel AI SDK + OpenAI/Anthropic | Pay per use |
| Hosting | Vercel | Free tier |
| i18n | next-intl (6 languages) | Free |
| CI/CD | GitHub Actions | Free tier |
| Testing | Playwright | Free |

## 🌍 Languages Supported

English · 简体中文 · 繁體中文 · Español · 한국어 · 日本語

## 📖 Features

- **AI Bible Tutor** — Ask questions about any passage, get context and cross-references
- **Structured Courses** — From Genesis to Revelation, beginner to advanced
- **Daily Devotional** — AI-generated reflections grounded in Scripture
- **Study Groups** — Learn together with believers worldwide
- **Prayer Wall** — Share requests, pray for one another
- **100% Free** — "Freely you have received; freely give" (Matthew 10:8)

## 🚀 Getting Started

```bash
git clone https://github.com/lesliezhili/emmausai.git
cd emmausai
npm install
cp .env.example .env.local
# Fill in your Supabase + AI API keys
npm run dev
```
